# U.S. Airline Delay & Performance Analytics Dashboard

## Project Overview

This project analyzes U.S. airline operational performance, weather-related delays, cancellation trends, and on-time efficiency using Power BI and business intelligence techniques.

The dashboard was designed to provide interactive insights into airline operations and identify factors contributing to delays and cancellations.

## Key Features

* Interactive KPI performance monitoring
* Weather impact analysis
* Airline operational efficiency comparison
* Delay trend visualization
* State-level delay analysis
* Dynamic filtering and slicers

## Key Insights

* Weather conditions significantly impacted airline delays and cancellations
* Delay patterns varied across airlines and departure hours
* Certain airlines demonstrated stronger operational efficiency and lower delay impact scores
* Early morning flights showed lower average delay times compared to peak operational periods

## Tools & Technologies

* Power BI
* Data Visualization
* Business Intelligence
* KPI Analysis
* Interactive Dashboarding
